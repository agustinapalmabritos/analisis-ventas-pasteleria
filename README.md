# Análisis de Ventas de Pastelería

## Descripción
Este proyecto analiza las ventas de una pastelería utilizando Excel y tablas dinámicas para identificar patrones de ventas y oportunidades de mejora.

## Objetivos
- Identificar productos más vendidos
- Analizar ventas por canal
- Analizar formas de pago
- Analizar ventas mensuales
- Generar conclusiones y recomendaciones

## Herramientas utilizadas
- Excel
- Tablas dinámicas
- Análisis de datos

## Resultados principales
- Cupcakes y cookies son los productos más vendidos
- El canal local concentra la mayoría de las ventas
- El efectivo es la forma de pago más utilizada
